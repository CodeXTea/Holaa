# Laman Prank Ringkas

Langkah:

1. Buat repo baru di GitHub (contoh: `prank-site`).
2. Salin fail `index.html`, `style.css`, `script.js` dan `README.md` ke repo dan commit ke branch `main`.
3. Pergi ke Settings -> Pages (atau Pages di sidebar) -> Source: pilih `main` branch, folder `/ (root)` -> Save.
4. Tunggu beberapa saat (kadang 1-2 min). Laman anda akan aktif di `https://<username>.github.io/<repo>/`.

Nota: Gunakan prank ini secara bertanggungjawab. Jangan gunakan untuk menipu, menakutkan, atau menipu maklumat sensitif orang lain.
